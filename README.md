
# Ferilskrá — statísk vefsíða

Þetta er einfalt, móttækilegt snið fyrir ferilskrá.

Skrár:

- `index.html` — aðalsíða ferilskrár
- `styles.css` — stílar
- `script.js` — smá hjálparforrit (afrita netfang, prenta/sækja PDF)

Hvernig á að keyra staðbundið:

Opnið `index.html` í vafra (tvöfaldur smellur eða `file:///`), eða keyrið einfaldan HTTP-þjón:

```
# Python 3
python3 -m http.server 8000
# opna síðan http://localhost:8000
```

Notkun:

- Hnapparnir „Sækja PDF" opna prentvalmynd vafrans (veljið „Vista sem PDF").
- Hnappurinn „Afrita netfang" afritar netfang í skyndiminni.

Skiptið út dæmiupplýsingum í `index.html` fyrir ykkar nafn, reynslu, tengla og upplýsingar.
